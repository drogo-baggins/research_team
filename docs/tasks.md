# Research Team Agent — ユーザーストーリーWBS

> 各ユーザーストーリーは独立した価値単位。未実装タスクは「✅ 実装済」「⚠️ 部品あり・未接続」「❌ 未実装」の3段階で表記。

## カテゴリ: UX

### US-1: 調査開始前にテーマと深さをユーザーと確認できる

**As a** ユーザー  
**I want to** 調査を始める前に規模感や方向性を擦り合わせたい  
**So that** 的外れな調査に時間を費やさない

| # | タスク | 状態 |
|---|---|---|
| 1-1 | CSM が冒頭でテーマを受け付ける (`run_interactive`) | ✅ 実装済 |
| 1-2 | 深さオプション (`quick` / `standard` / `deep`) を CLI で指定可能 | ✅ 実装済 |
| 1-3 | CSM がテーマと深さをユーザーに確認し、合意を取ってから調査開始する対話フロー | ✅ 実装済 |
| 1-4 | 参照ファイル（テキスト・Excel）をプロジェクトフォルダ経由で調査入力として渡せる | ❌ 未実装（US-3 完了後に実装。エンジン部分は実装済み） |
